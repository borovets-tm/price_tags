Инструкция по обновлению цен после прихода товара

Шаг 1.
Откройте файл с ценами прихода.
Шаг 2.
Выделите заполненные строки и столбцы таблицы и нажмите Ctrl+C
Шаг 3.
Вставьте скопированное содержимое на новый лист. Удалите лист 1 (ПКМ -> Удалить лист)
Шаг 4.
Если есть столбец со штрихкодами, удалите его и переименуйте столбцы (первая строка) по следующим критериями:
1. Item -> sku
2. Столбец с ценами -> price
3. Столбец с комментариями -> old_price
4. Следующий пустой столбец -> is_red_price
После этого выделите столбец с ценами, нажмите Ctrl+1. Выберите вкладку "Число", в ней выберите формат "Числовой" и снимите галочку "Разделитель групп разрядов()"
Заполните столбцы по следующим правилам:
1. old_price - если указана вторая цена, то введите ее, если нет, то поставьте 0. Следите за тем, чтобы 4-х значные цифры имели формат, как на фото, при необходимости повторите шаг выше для данного столбца.
2. is_red_price = если ценник красный - 1, если черный - 0
Шаг 5.
Сохраните файлЖ: Сохранить как -> формат CSV, как на фото
Шаг 6.
Итоговый файл должен выглядеть вот так. Загрузите файл.
